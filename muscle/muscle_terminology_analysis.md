### ⚠️ 需要改进的名称

**头颈部改进建议**
- "颈浅" → "颈浅层肌" 或 "颈浅肌群" (Superficial neck muscles)
- "中层" → "舌骨上肌群" 和 "舌骨下肌群" (Suprahyoid & Infrahyoid muscles)
- "深层" → "颈深层肌" (Deep neck muscles)

**躯干部改进建议**
- "背浅" → "背浅层肌" (Superficial back muscles)
- "背深" → "背深层肌" (Deep back muscles)
- "胸上肢肌" → "胸上肢肌群" (Muscles of pectoral girdle)
- "胸固有肌" → "胸固有肌群" 或 "胸壁肌" (Thoracic wall muscles)
- "腹肌-前外侧" → "腹前外侧肌群" (Anterolateral abdominal muscles)
- "腹肌-后群" → "腹后群肌" (Posterior abdominal muscles)

**上肢改进建议**
- "上肢带肌" → "肩带肌" (Muscles of shoulder girdle)
- "上臂肌-屈" → "上臂屈肌群" (Arm flexors)
- "上臂肌-伸" → "上臂伸肌群" (Arm extensors)
- "前臂肌-屈浅" → "前臂浅层屈肌" (Superficial forearm flexors)
- "前臂肌-屈深" → "前臂深层屈肌" (Deep forearm flexors)
- "前臂肌-伸浅" → "前臂浅层伸肌" (Superficial forearm extensors)
- "前臂肌-伸深" → "前臂深层伸肌" (Deep forearm extensors)

**下肢改进建议**
- "下肢带肌-前" → "髋肌前群" (Anterior hip muscles)
- "下肢带肌-后" → "髋肌后群" (Posterior hip muscles)
- "大腿前" → "大腿前群肌" (Anterior thigh muscles)
- "大腿后" → "大腿后群肌" (Posterior thigh muscles)
- "大腿内侧" → "大腿内侧肌群" (Medial thigh muscles)
