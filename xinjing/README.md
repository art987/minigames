# 瑞文智力测试 - 纯前端版本

## 项目简介
这是一个完全改造为纯前端版本的瑞文标准推理测验(SPM)系统。无需后台、无需登录、无需付费,所有功能都在本地完成。

## 项目特点
- ✅ **纯前端**: 无PHP后台,完全使用HTML+JavaScript实现
- ✅ **无登录**: 无会员系统,无需注册登录
- ✅ **免费使用**: 无付费功能,完全免费开放
- ✅ **本地数据**: 所有题目数据存储在本地JS文件中
- ✅ **完整测试**: 包含完整的72题瑞文智力测试
- ✅ **实时计算**: 测试完成后即时计算IQ值

## 使用方法

### 1. 本地运行
直接用浏览器打开 `index.html` 即可开始测试,或使用本地服务器:

```bash
# 使用Python启动本地服务器
python -m http.server 8080

# 或使用Node.js
npx http-server -p 8080
```

然后访问: http://localhost:8080

### 2. 开始测试
1. 打开测试页面,阅读测试说明
2. 点击"开始智商测试"按钮
3. 逐题作答,可随时返回上一题
4. 完成所有72题后,输入年龄
5. 点击"查看结果"获取IQ值

### 3. 查看结果
- 系统会显示原始得分(答对题数)
- 根据年龄和分数自动计算IQ值
- 显示IQ等级(超优/优秀/中上/中等/中下/临界/偏低)
- 可查看换算标准表图片

## 技术说明

### 前端技术栈
- **HTML5**: 页面结构
- **CSS3**: 样式设计(响应式布局)
- **JavaScript**: 业务逻辑和数据计算
- **LocalStorage**: 本地数据存储

### 数据组织
- **题目数据**: 每题独立JS文件,按组分类存储
- **答案数据**: 嵌入在各题目JS文件中
- **IQ换算**: 基于年龄和原始分数的换算算法

### 功能特点
1. **进度显示**: 实时显示答题进度条
2. **答案保存**: 自动保存用户答案到LocalStorage
3. **即时计算**: 无需后台,前端直接计算IQ值
4. **响应式设计**: 支持手机、平板、电脑等设备

## 测试内容

### 题目分组
- **A组**: 12题,每题6个选项(基础推理)
- **B组**: 12题,每题6个选项(类比推理)
- **AB组**: 12题,每题6个选项(递进推理)
- **C组**: 12题,每题8个选项(拓扑推理)
- **D组**: 12题,每题8个选项(拓扑类比)
- **E组**: 12题,每题8个选项(高阶推理)

### 换算说明
IQ值根据年龄段和原始分数换算:
- 年龄段选择: 按就近原则(如22岁选20岁年龄段)
- 分数匹配: 按就近最低原则(如62分选64分的对应值)
