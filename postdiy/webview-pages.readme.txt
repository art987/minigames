这个文件是“远程配置模板说明版”，给不会编程的人看。

先记住 3 条最重要的规则：
1. 真正放到服务器给 App 读取的内容，最稳妥的是“纯 JSON”，不要把这些说明文字一起传上去。
2. 你可以直接参考同目录下的 `webview-pages.remote.sample.json`，那一份就是可以直接部署的纯数据示例。
3. App 现在默认会请求：`https://peacelove.top/postdiy/webview-pages.uts`

你只需要改下面这些常见位置：
- 想改首页网址：改 `tabBarItems` 里首页那一项的 `webviewUrl`
- 想控制某页是否下拉刷新：改 `enablePullDownRefresh`
- 想改底部图标：改 `activeIconUrl` 和 `inactiveIconUrl`
- 想改加载图：改 `webviewLoadingStyle.remoteSplashImageUrl`
- 想改加载文案：改 `webviewLoadingStyle.loadingText`

布尔值说明：
- `true` = 开启 / 允许
- `false` = 关闭 / 不允许

推荐你的实际操作方式：
1. 打开 `config/webview-pages.remote.sample.json`
2. 只改你想改的值
3. 把那份纯 JSON 文件上传到你的服务器地址
4. 保证服务器返回的是纯文本 JSON，不要自动插入 HTML 页面壳子
