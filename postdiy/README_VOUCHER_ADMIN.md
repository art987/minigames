# 会员激活码管理系统

## 系统简介

会员激活码管理系统是一个专为管理员设计的后台系统，用于生成、管理和查看VIP会员激活码。

## 功能特性

- 🔐 **管理员登录**：仅允许指定用户ID登录
- 📋 **激活码生成**：支持生成不同有效期的激活码
- 📊 **激活码管理**：查看所有激活码的状态和详情
- 🔄 **云端同步**：所有数据存储在云端，确保数据安全

## 如何使用

### 1. 访问管理系统

打开浏览器，访问：
```
http://localhost:8080/voucher-admin.html
```

### 2. 登录系统

输入上述登录信息，点击「登录」按钮。

### 3. 生成激活码

- 在「生成新激活码」部分，选择有效期（1个月、3个月、6个月、1年、2年）
- 输入生成数量（1-100个）
- 点击「生成激活码」按钮
- 系统会在本地生成符合要求的激活码，然后上传到云端存储

### 4. 查看激活码列表

系统会自动加载所有激活码，包括：
- 激活码编号
- 状态（未使用/已使用）
- 有效期
- 创建时间
- 使用信息（如果已使用）

## 激活码规则

- 格式：`VIP` + 8位随机字符
- 不包含数字"4"
- 至少包含1个"6"和2个"8"
- 支持的有效期：1个月、3个月、6个月、1年、2年

## 技术实现

- **前端**：HTML5 + CSS3 + JavaScript
- **后端**：腾讯云 CloudBase 云函数
- **存储**：腾讯云数据库

## 云函数说明

### 已部署的云函数

1. **admin-generate-vouchers**：批量生成激活码
2. **admin-manage-vouchers**：管理激活码（查看列表、删除等）

### API 接口

- **生成激活码**：`POST https://api.peacelove.top/admin-generate-vouchers`
- **管理激活码**：`POST https://api.peacelove.top/admin-manage-vouchers`

## 注意事项

1. 请妥善保管登录密码
2. 生成的激活码请妥善保存，避免泄露
3. 定期查看激活码使用情况，及时清理过期或已使用的激活码
