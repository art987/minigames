## 软件用途

闪喵海报是一款面向中小商家、个体工商户的节日海报快速生成工具。软件采用Hybrid混合开发模式（uni-app x App壳 + H5业务页面），用户可通过选择模板、填写商家信息、编辑促销文案，一键生成适合朋友圈传播的宣传海报。

主要功能包括：
- 节日模板自动推荐
- 品牌信息编辑（Logo、名称、二维码）
- 促销文案编辑和行业文案选择
- 海报生成和保存到相册
- VIP会员购买和权限管理
- 邀请分享功能

## 技术特点

1. **混合架构**：原生App壳负责底层能力，H5负责业务逻辑，通过JSBridge双向通信
2. **远程配置**：页面地址、导航栏、更新提示等均可远程动态调整
3. **多端适配**：支持Android和iOS平台，兼容WebView和浏览器环境
4. **云服务**：使用腾讯云CloudBase提供数据存储、云函数、CDN加速
