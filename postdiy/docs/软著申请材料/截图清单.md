# APP截图清单

软著申请要求提供APP在手机上的运行截图，**必须是真机或模拟器截图，顶部需显示状态栏**（电量、时间），以证明是APP环境而非浏览器。

---

## 截图要求

### 基本要求：
1. **必须是APP环境**：不能是浏览器打开网页的截图
2. **顶部必须有状态栏**：显示电量、时间、信号等
3. **底部可有导航栏**：显示APP的底部导航
4. **分辨率**：建议使用1080x1920（竖屏）
5. **格式**：PNG或JPG，清晰度足够

### 建议工具：
- **Android Studio模拟器**：推荐Pixel 5模拟器
- **真机截图**：音量减+电源键
- **iOS模拟器**：Command+S截图

## 如何获取APP截图

### 方法一：使用Android Studio模拟器

1. 打开Android Studio
2. 创建Pixel 5模拟器（Android 11）
3. 安装Uni-app打包的APK
4. 打开APP，导航到各个页面
5. 点击模拟器右侧的"截图"按钮

### 方法二：使用真机

1. 用HBuilderX云打包生成APK/IPA
2. 安装到真机
3. 打开APP，导航到各个页面
4. 按音量减+电源键截图
5. 从相册导出截图

### 方法三：使用Uni-app调试

1. HBuilderX连接真机
2. 使用"运行-运行到手机或模拟器"
3. 在调试模式下截图

## 注意事项

### ✅ 正确的截图：
- 顶部有状态栏（电量、时间、信号）
- 底部有导航栏（如有）
- 是APP运行环境，不是浏览器

### ❌ 错误的截图：
- 浏览器打开网页的截图
- 没有状态栏的截图
- 截图模糊不清晰
