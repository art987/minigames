## 五、关键代码示例

### 1. App.uvue（应用入口）

```vue
<script>
export default {
    onLaunch: function() {
        console.log('App Launch');
        // 初始化全局状态
        this.initGlobalState();
    },
    onShow: function() {
        console.log('App Show');
    },
    onHide: function() {

            if (userId) {
                this.globalData.isLoggedIn = true;
            }
        }
    }
}
</script>
```

### 2. bridge-webview.uvue（WebView宿主）

```vue
<template>
    <view class="webview-container">
        <web-view
            :src="currentUrl"
            @message="handleMessage"
            @error="handleError"
            @load="handleLoad"
        ></web-view>
    </view>
</template>

<script>
export default {
    data() {
        return {
            currentUrl: 'https://peacelove.top/postdiy/index.html'
        }
    },
    methods: {
        handleMessage(e) {
            const data = e.detail.data[0];
            const action = data.action;

            if (action === 'downloadImage') {
                this.saveImageToPhotosAlbum(data.dataUrl, data.fileName);
            } else if (action === 'copyText') {
                uni.setClipboardData({
                    data: data.text,
                    success: () => {
                        uni.showToast({ title: '复制成功' });
                    }
                });
            } else if (action === 'openPaymentUrl') {
                uni.navigateTo({
                    url: '/pages/payment/index?url=' + encodeURIComponent(data.url)
                });
            }
        },
        saveImageToPhotosAlbum(dataUrl, fileName) {
            // base64图片保存逻辑
            const fs = uni.getFileSystemManager();
            const filePath = `${wx.env.USER_DATA_PATH}/${fileName}`;

            fs.writeFile({
                filePath: filePath,
                data: dataUrl.replace(/^data:image\/\w+;base64,/, ''),
                encoding: 'base64',
                success: () => {
                    uni.saveImageToPhotosAlbum({
                        filePath: filePath,
                        success: () => {
                            uni.showToast({ title: '保存成功' });
                        }
                    });
                }
            });
        },
        handleError(e) {
            console.error('WebView加载失败:', e);
            // 显示错误遮罩
            this.showErrorOverlay = true;
        }
    }
}
</script>
```

### 3. webview-pages.uts（远程配置）

```typescript
export function loadRemoteConfig(): Promise<WebViewConfig> {
    return new Promise((resolve) => {
        const defaultConfig = getLocalConfig();
        const timeout = 3000;

        const timer = setTimeout(() => {
            resolve(defaultConfig);
        }, timeout);

        uni.request({
            url: 'https://peacelove.top/postdiy/webview-pages.remote.json',
            success: (res) => {
                clearTimeout(timer);
                if (res.statusCode === 200 && res.data) {
                    const remoteConfig = res.data as WebViewConfig;
                    const mergedConfig = mergeConfig(defaultConfig, remoteConfig);
                    resolve(mergedConfig);
                } else {
                    resolve(defaultConfig);
                }
            },
            fail: () => {
                clearTimeout(timer);
                resolve(defaultConfig);
            }
        });
    });
}
```

## 七、注意事项

### ✅ 必须做的：
1. **保留业务变量名**：如 `handleUserLogin()`、`autoSelectByDate()`
2. **保留注释**：体现开发思路
3. **保留函数命名**：体现功能模块

1. **不要混淆变量名**：如 `a`、`b`、`c` 等无意义命名
