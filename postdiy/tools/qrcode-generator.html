常见软件二维码获取方法

个人微信二维码

  1. 打开微信，点击右下角「我」
  2. 点击你的头像进入个人信息页面
  3. 点击「二维码名片」
  4. 长按二维码图片，选择「保存图片」

微信群二维码

  1. 打开微信群聊窗口
  2. 点击右上角「...」进入群设置
  3. 点击群名称下方的二维码图标
  4. 长按二维码图片，选择「保存图片」
  5. 注意：微信群二维码有效期为7天

微信公众号二维码

  1. 打开微信公众号后台
  2. 点击「设置与开发」-「公众号设置」
  3. 在「帐号详情」中找到二维码
  4. 点击「下载二维码」保存到本地

QQ二维码

  1. 打开QQ，点击左上角头像
  2. 点击你的头像进入个人资料页面
  3. 点击「我的二维码」
  4. 点击右上角「...」，选择「保存到手机」

支付宝二维码

  1. 打开支付宝，点击「收钱」
  2. 点击二维码下方的「保存图片」
  3. 如需付款码，点击「付款」，截图保存

微博二维码

  1. 打开微博，点击右下角「我」
  2. 点击你的头像进入个人主页
  3. 点击「二维码」图标
  4. 长按二维码图片，选择「保存图片」
