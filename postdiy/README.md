# 朋友圈海报制作中心 - 在线海报设计工具

一款功能强大的在线海报设计工具，帮助用户快速创建、编辑和定制节日主题及行业促销海报。支持微信浏览器内访问，具备完整的VIP系统、券码验证和丰富的贴纸素材库。

## 🎯 核心功能

### 海报编辑
- **实时编辑**：商家信息（名称、Logo、二维码、促销文案）实时预览
- **模板切换**：200+节日模板，涵盖传统节日、节气、西方节日
- **背景上传**：支持自定义背景图片上传
- **贴纸系统**：10+贴纸分类（遮罩、花朵、好吃、好看、早安短句等），支持点击直接添加
- **导出下载**：高清PNG格式导出，支持本地保存

### 行业促销模板
- **通用类目**：20+适用于各类场景的促销文案模板
- **行业分类**：奶茶、餐厅、少儿画室、服装店、美容美发等10+行业专属模板
- **智能文案**：每条模板包含引导语、福利信息、地址/联系方式

### VIP系统
- **用户登录**：支持VIP账号登录（预设百事可乐、可口可乐、星巴克、麦当劳等）
- **用户信息管理**：显示用户ID、有效期、账户类型
- **显示管理**：控制海报元素显示/隐藏
- **券码验证**：4位数字券码验证，支持不同有效期

### 智能建议
- **每日提示**：根据日期和时间提供当日制作建议
- **节日提醒**：显示今日节日、未来节日倒计时
- **时间分类**：早安（9:30前）、日常、晚安（9:30后）不同时段建议

## 📱 手机端特性

### 触摸交互
- **长按提示**：区域长按显示白色虚线框和发光效果（3秒自动消失）
  - 左侧：更换模板
  - 右上：相册上传
  - 右下：拍照上传
- **点击添加**：贴纸直接点击即可添加到画布
- **预览查看**：贴纸可查看大图，支持滑动切换

### 微信适配
- 微信浏览器内自动检测并提示
- 右上角"..."选择外部浏览器获取完整功能

## 🏗️ 技术架构

### 前端技术栈
- **HTML5**：响应式页面结构
- **CSS3**：动画效果、Flexbox布局、渐变效果

## 🚀 快速开始

### 本地运行

1. **启动HTTP服务器**
   ```bash
   cd postdiy
   python -m http.server 8000
   # 或使用 Node.js
   npx http-server -p 8000
   ```

2. **访问网站**
   - 桌面端：`http://localhost:8000`
   - 手机端：`http://<你的IP>:8000`
   - 微信调试：`http://localhost:8000?browser=wechat`

3. **测试账号**
   - ID: `pepsi` | 密码: `123456`
   - ID: `cocacola` | 密码: `vip1234`
   - ID: `starbucks` | 密码: `vip2026`
   - ID: `mcdonalds` | 密码: `vip1234`

4. **券码测试**
   - `8866`：1个月有效期
   - `9988`：3个月有效期
   - `1016`：9个月有效期
   - `9999`：12个月有效期

## 📖 使用指南

### 首页 (index.html)
1. 浏览分类模板（按月份或节日）
2. 点击模板进入编辑器
3. 点击"每日建议"获取制作建议

### 编辑器 (editor.html)

#### 基础编辑
1. **修改商家信息**
   - 点击Logo区域上传图片
   - 点击二维码区域上传图片
   - 点击文案区域编辑促销信息

2. **更换模板**
   - 点击"选择模板"按钮
   - 浏览节日模板
   - 点击模板立即应用

3. **上传背景**
   - 长按右侧区域（左：更换模板，右上：相册，右下：拍照）
   - 或点击上传按钮

4. **添加贴纸**
   - 点击"贴纸"按钮
   - 选择分类（遮罩、花朵、好吃、好看、早安短句等）
   - 点击贴纸直接添加到画布
   - 可拖拽、缩放、旋转贴纸

5. **导出海报**
   - 点击"保存海报"按钮
   - 等待截图生成
   - 点击下载保存到手机

#### 高级功能
- **显示管理**：控制Logo、二维码、促销文案等元素显示/隐藏
- **VIP功能**：登录后可使用更多高级功能
- **券码验证**：输入券码延长使用期限

## 📱 手机端长按功能

### 触摸区域说明
1. **左侧区域**（`.trigger-left`）
   - 长按提示："更换模板"
   - 功能：打开模板选择弹窗
   - 效果：白色虚线框 + 发光动画（3秒）

2. **右上区域**（`.trigger-top`）
   - 长按提示："相册上传"
   - 功能：触发相册选择
   - 效果：白色虚线框 + 发光动画（3秒）

3. **右下区域**（`.trigger-bottom`）
   - 长按提示："拍照上传"
   - 功能：触发相机拍照
   - 效果：白色虚线框 + 发光动画（3秒）

### 贴纸交互
- **点击贴纸**：直接添加到画布，自动关闭弹窗
- **点击"查看"按钮**：打开预览弹窗查看大图
- **预览区域点击**：添加贴纸并关闭弹窗
- **拖拽调整**：长按拖拽调整位置
- **缩放旋转**：使用控制点调整大小和角度

## 🔧 开发说明

### 代码规范
- 模块化设计，避免全局变量污染
- 使用 `window.` 前缀存储全局对象
- 事件监听器使用 `addEventListener`
- CSS动画使用 `@keyframes`

### 性能优化
- 动态加载模板数据（防止缓存）
- 图片懒加载
- 贴纸资源预加载
- html2canvas 高质量渲染

- Safari 14+
- Edge 90+
